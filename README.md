\# FYP: TikTok 文化遗产治理研究



\## 项目目的

研究 TikTok 国际版政策文件 2020-2025 年的演变，

分析其中关于文化、传统、宗教、原住民内容条款的变化。



\## 指导老师

郭倩(Dr. Qian Guo), BNBU



\## 项目结构

\- data/raw/  原始爬取的 HTML 文件

\- data/processed/  清洗后的文本数据

\- code/python/  Python 脚本（爬虫、Gemini API 调用、清洗）

\- code/r/  R 脚本（词汇分析、网络分析、可视化）

\- codebook/  编码手册

\- results/figures/  输出的图

\- results/tables/  输出的表

\- writing/  论文草稿

\- notes/  阅读笔记和工作日志
